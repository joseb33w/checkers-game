# Checkers Game

A simple, single-file checkers game in pure HTML/CSS/JS.

## Modes
- **vs Computer** — play against an AI opponent
- **vs Friend** — local two-player on the same device

## Features
- Standard checkers rules (mandatory captures, chain jumps, king promotion)
- Movable pieces highlighted with a gold ring
- Capture targets shown as teal rings
- Undo support
- Mobile-friendly responsive board

## Run
Just open `index.html` — everything is self-contained.
